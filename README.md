# GitHub_Followers
GitHub_Followers is an iOS app, which allows you to search for a GitHub user and get all his followers.

## Features
* 100% UIKit
* 100% Programmatic UI
* No storyboard/xib
* No 3rd party libraries
* Minimum Deployments: iOS 13.2
